### Installation
```
# clone the repo
$ git clone https://github.com/robert-choi/ml-doodle-jump.git

# change the working directory
$ cd ml-doodle-jump

# install python and pip if they are not installed

# install the requirements
$ pip install -r requirements.txt
```
